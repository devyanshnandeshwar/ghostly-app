# AI Model

This repository contains the AI model used by the Ghostly app for Gender and Age detection.

## Objective

To build a gender and age detector that can approximately guess the gender and age of the person (face) in a picture or through the webcam.

## Usage

The model exposes an endpoint for inference which the Ghostly server consumes.

## Dependencies

- OpenCV
- argparse

Use the provided `requirements.txt` to install dependencies.
